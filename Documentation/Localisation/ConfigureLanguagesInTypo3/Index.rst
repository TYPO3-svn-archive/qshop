Configure languages in TYPO3
----------------------------

If you want to deliver your website in more than one language you have to configure TYPO3.

Please refer to this documents:

- TYPO3 Localization Guide
  http://typo3.org/documentation/document-library/core-documentation/doc_l10nguide/current/
- Browser Tutorial Localization (en)
  http://typo3.org/extensions/repository/view/browser_tut_localization_en/current/
