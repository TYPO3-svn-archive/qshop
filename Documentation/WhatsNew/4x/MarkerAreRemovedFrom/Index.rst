Marker are removed from TypoScript
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

[4.0.3]

Marker are removed from the TypoScript and are replaced by the field names.

Former TypoScript snippet:

value = ###TX_QUICKSHOP_PRODUCT.TITLE###

Current TypoScript

field = tx_quickshop_product.title
