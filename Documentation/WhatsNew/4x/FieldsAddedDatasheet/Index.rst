Fields added: datasheet, dimension, ean, in stock amount, material
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

[4.0.0]

Items have the new fields:

- datasheet for displaying item data in a table.
- dimension category dimension / size
- in stock amount the amount of the current item in stock
- material category material

[4.0.5]

ean: European Article Number
