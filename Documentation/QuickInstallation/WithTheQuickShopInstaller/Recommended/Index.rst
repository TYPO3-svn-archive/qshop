Recommended
^^^^^^^^^^^

- Install the Quick Shop Installer (quickshop_installer)
  http://typo3.org/extensions/repository/view/quickshop_installer/
- Add the plugin Quick Shop Installer to a page and enable it.
- Load the page in the frontend.
- Click "Install it!"
